## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/Bhayal07/Project-Managment-Tool.git
   cd Project-Managment-Tool
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

3. Run the app:

   ```sh
   npm start
   ```

   Or if it's a full stack app:

   ```sh
   cd server
   npm install
   npm start

   cd ../client
   npm install
   npm start
   ```
